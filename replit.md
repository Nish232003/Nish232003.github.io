# Nishita Jain Portfolio Website

## Overview
A creative portfolio website for Nishita Jain, a Data Scientist and Artist. The website showcases her unique blend of data science expertise and artistic talents.

## Project Structure
```
/
├── index.html          # Main HTML structure
├── style.css           # Complete styling with pastel theme and dark mode
├── script.js           # JavaScript for interactivity
├── assets/             # Directory for resume and other assets
└── replit.md           # Project documentation
```

## Features
- **Hero Section**: Animated intro with floating shapes
- **About Section**: Bio with placeholder for profile image
- **Data Projects**: Grid layout with filter functionality, includes placeholder for live dashboard embeds
- **Art Gallery**: Masonry layout with placeholders for paintings and sketches
- **Skills**: Visual progress bars for technical, artistic, and soft skills
- **Experience Timeline**: Education, work experience, and certifications
- **Blog Section**: Links to Medium articles
- **Contact Section**: Contact form with mailto integration
- **Dark/Light Mode Toggle**: Persistent theme preference
- **Responsive Design**: Mobile-friendly layout

## Technology Stack
- HTML5
- CSS3 (CSS Variables for theming)
- Vanilla JavaScript
- Font Awesome icons
- Google Fonts (Playfair Display, Inter)

## Color Scheme (Pastel)
- Light Mode: Soft pinks, lavenders, mint
- Dark Mode: Deep blues with pastel accents

## User Preferences
- Aesthetic pastel colors
- Dark/light mode toggle
- Blog section included
- Animated transitions
- Placeholder images (to be updated by user)
- Resume download option

## How to Update
1. **Profile Image**: Replace the placeholder in About section with actual image
2. **Artworks**: Replace placeholders in Art Gallery section with artwork images
3. **Projects**: Update project cards with actual screenshots and links
4. **Resume**: Add resume PDF to assets/resume.pdf
5. **Live Dashboards**: Embed Tableau/Streamlit apps in the designated embed placeholder
